# Messaging Matrix Template

## Purpose

Create a comprehensive framework that maps key messages, proof points, and value propositions across different audience segments and communication channels. This ensures consistent, targeted messaging while allowing for appropriate customization by audience and medium.

## Ownership & Collaboration

- **Document Owner**: Content Strategist / Brand Manager / Product Marketing
- **Contributor Roles**: Campaign Manager, Product Marketing, Sales Enablement, PR Manager
- **Reviewers**: Marketing Director, Executive Leadership
- **Automation Inputs**: Campaign strategy, audience research, product positioning, competitive analysis
- **Automation Outputs**: `messaging-matrix-{campaign-or-product}.md` with complete message mappings

## Completion Checklist

- [ ] Core value proposition clearly articulated
- [ ] Key messages defined for each audience segment
- [ ] Proof points substantiated with data or evidence
- [ ] Channel-specific message variations documented
- [ ] Tone and voice guidance provided
- [ ] Messaging hierarchy established (primary, secondary, tertiary)
- [ ] Competitive differentiation clear
- [ ] Messaging tested with sample audiences or stakeholders
- [ ] All stakeholders aligned on messaging framework

## Agent Notes

### For Content Strategist

- Ground all messaging in customer research and data, not assumptions
- Test messaging with sample audiences before broad rollout
- Provide specific examples, not just abstract guidelines
- Maintain consistency while allowing channel-appropriate variation
- Update messaging based on performance data and feedback

### For Campaign Manager

- Ensure all campaign assets follow this messaging framework
- Provide this matrix to all content creators upfront
- Review content against messaging matrix before approval
- Track which messages resonate best and optimize
- Escalate messaging conflicts or confusion for resolution

### For Sales Enablement

- Train sales team on audience-specific messaging
- Provide battle cards for competitive situations
- Collect sales feedback on objections and refine responses
- Update discovery questions based on messaging priorities
- Share customer language to inform authentic messaging

### For PR Manager

- Use media talking points for all spokesperson prep
- Ensure press releases align with core positioning
- Adapt messaging for journalist audiences (newsworthy angle)
- Track media coverage to see which messages land
- Update spokesperson quotes based on what gets picked up

### For Product Marketing

- Validate that product features align with benefits messaging
- Provide proof points and customer evidence
- Flag when product changes require messaging updates
- Ensure competitive differentiation claims are accurate
- Collaborate with sales on battle cards and objection handling
