## Purpose

This template provides a structured outline for podcast episodes, including intro/outro scripts, segment breakdowns, guest questions, and production notes. Suitable for interview shows, solo commentary, panel discussions, and narrative storytelling formats.

## Production Checklist

### Pre-Recording

**Technical Setup**:
- [ ] Recording platform tested (Zoom, Riverside, SquadCast, etc.)
- [ ] Microphones tested (host and guest)
- [ ] Headphones confirmed (prevent echo/feedback)
- [ ] Backup recording setup (local recording + cloud)
- [ ] Internet connection stable (wired preferred over WiFi)

**Guest Prep**:
- [ ] Guest sent tech check instructions 24 hours in advance
- [ ] Guest sent outline/questions 48-72 hours in advance (optional)
- [ ] Guest confirmed recording time (with timezone confirmation)
- [ ] Guest reminded 1 hour before recording
- [ ] Guest release form signed

**Host Prep**:
- [ ] Episode outline reviewed
- [ ] Questions finalized (but stay flexible)
- [ ] Guest bio and background researched
- [ ] Water nearby (for host and guest if in-person)

### Recording

**Recording Best Practices**:
- [ ] Record in quiet space (minimal background noise)
- [ ] Use pop filter (reduce plosives)
- [ ] Maintain consistent mic distance (6-8 inches)
- [ ] Monitor levels (peak at -6dB to -12dB)
- [ ] Record separate tracks for host and guest (easier editing)
- [ ] Take notes on timestamps for great quotes/moments

**During Recording**:
- [ ] Intro music and script
- [ ] Main interview content
- [ ] Lightning round (if applicable)
- [ ] Outro and CTAs
- [ ] Room tone recording (30 seconds of silence for editing)

**Recording Time Buffer**:
- Allocate 1.5x target length (e.g., 60 minutes for 40-minute episode)
- Allow time for small talk, tech issues, retakes

### Post-Recording

**Immediate Actions**:
- [ ] Download/backup raw audio files
- [ ] Send thank-you email to guest
- [ ] Collect guest headshot and bio for show notes
- [ ] Note any specific timestamps or edits needed

**Editing**:
- [ ] Remove filler words (um, uh, like) - but don't over-edit
- [ ] Cut awkward pauses, false starts, off-topic tangents
- [ ] Add intro/outro music
- [ ] Add sponsor messages (if applicable)
- [ ] Normalize audio levels (host and guest balanced)
- [ ] Add EQ/compression for professional sound
- [ ] Export final file (MP3, 128-192 kbps, mono or stereo)

**Show Notes & Metadata**:
- [ ] Write episode description (300-500 words with keywords)
- [ ] Create timestamps for key moments
- [ ] List resources mentioned (links, books, tools)
- [ ] Add guest links (website, social, company)
- [ ] Upload episode art (1400x1400px minimum)

**Quality Assurance**:
- [ ] Listen to full episode (check for audio issues, awkward edits)
- [ ] Verify intro/outro music levels
- [ ] Confirm all sponsor messages included
- [ ] Check file metadata (title, description, tags)

## Review Checklist

### Content Quality

- [ ] **Valuable Content**: Episode delivers actionable insights or entertaining stories
- [ ] **Pacing**: Good flow, no long tangents or dead air
- [ ] **Authenticity**: Conversation feels natural, not overly scripted
- [ ] **Balance**: Host doesn't dominate - guest speaks 60-70% of the time
- [ ] **Clarity**: Complex topics explained clearly for target audience
- [ ] **Engagement**: Compelling stories, examples, or data points included

### Audio Quality

- [ ] **Clarity**: Both host and guest audio clear and intelligible
- [ ] **Levels**: Balanced volume (host and guest equally audible)
- [ ] **Background Noise**: Minimal or no background noise, hum, echo
- [ ] **Music Mix**: Intro/outro music at appropriate level (not overpowering speech)
- [ ] **Editing**: Smooth cuts, no abrupt transitions or awkward silences
- [ ] **File Quality**: MP3 exported at 128-192 kbps (balance quality and file size)

### SEO & Metadata

- [ ] **Title**: Descriptive, keyword-rich, <60 characters
- [ ] **Description**: Detailed summary with keywords, timestamps, links (300-500 words)
- [ ] **Tags**: Relevant keywords/tags added
- [ ] **Show Notes**: Comprehensive show notes with resources, timestamps, guest links
- [ ] **Episode Art**: Custom episode art (1400x1400px, branded)
- [ ] **Transcript**: Transcript uploaded (for accessibility and SEO)

### Distribution Ready

- [ ] **RSS Feed**: Episode uploaded to podcast host (Libsyn, Buzzsprout, Anchor, etc.)
- [ ] **Platforms**: Distributed to Apple Podcasts, Spotify, Google Podcasts, etc.
- [ ] **Website**: Embedded player on website with show notes
- [ ] **YouTube**: Video version uploaded (if applicable)
- [ ] **Captions**: Captions/transcript available

## Agent Notes

### For Content Strategist
- Align podcast topics with broader content strategy and audience needs
- Book guests who provide unique insights and credibility
- Monitor episode performance and identify top-performing topics
- Coordinate cross-promotion with blog, email, social channels

### For Podcast Producer
- Prioritize audio quality (invest in good mics, quiet recording space)
- Keep conversations authentic and conversational (not overly scripted)
- Edit for clarity and pacing, but preserve natural speech patterns
- Create compelling show notes and promotional assets
- Maintain consistent publishing schedule (weekly, bi-weekly, monthly)

### For Copywriter (Show Notes & Descriptions)
- Write detailed, keyword-rich episode descriptions (300-500 words)
- Include timestamps for easy navigation
- List all resources, tools, and links mentioned
- Optimize for SEO (podcast episodes rank in Google search)
- Write compelling social copy that drives listens
