## Purpose

This template provides a structured approach to writing video scripts with timing, shot descriptions, and production notes. Suitable for explainer videos, product demos, testimonials, social media videos, and video ads.

## Script Structure

### Timing Convention

**Format**: `[MM:SS]` (minutes:seconds) for total runtime; each scene/shot includes duration estimate.

**Example**:
```
[00:00-00:05] SCENE 1: Opening hook (5 seconds)
[00:05-00:15] SCENE 2: Problem statement (10 seconds)
...
```

**Note**: Timing is approximate and will be refined in editing. Average speaking pace is 150 words/minute (2.5 words/second).

## Production Checklist

### Pre-Production

**Script Finalization**:
- [ ] Script reviewed by stakeholders (marketing, product, legal if needed)
- [ ] Voiceover artist or on-camera talent confirmed
- [ ] Timing validated (read script aloud, adjust as needed)

**Storyboard**:
- [ ] Storyboard created (optional but recommended for complex videos)
- [ ] Key frames sketched or described

**Shot List**:
- [ ] All required shots listed (on-camera, b-roll, screen recordings, animations)
- [ ] Locations scouted (if live action)
- [ ] Props/equipment list finalized

**Assets Gathered**:
- [ ] Product logos, brand assets
- [ ] Stock footage licensed (if needed)
- [ ] Music licensed (Artlist, Epidemic Sound, etc.)
- [ ] Sound effects library

**Talent & Crew**:
- [ ] Voiceover artist booked
- [ ] On-camera talent confirmed (if applicable)
- [ ] Videographer/director scheduled
- [ ] Editor assigned

### Production

**Filming**:
- [ ] All scenes filmed according to shot list
- [ ] Multiple takes captured for each scene (safety)
- [ ] Audio quality confirmed (no background noise, clear VO)
- [ ] Lighting consistent and professional

**Screen Recordings** (if applicable):
- [ ] Product demo recorded in HD (1080p min)
- [ ] Mouse movements smooth and intentional
- [ ] No typos, errors, or distracting elements on screen

**Animation/Graphics** (if applicable):
- [ ] Motion graphics created (After Effects, Premiere, etc.)
- [ ] Graphics aligned with brand guidelines (colors, fonts)

### Post-Production

**Editing**:
- [ ] Video cut to script timing (tight pacing, no dead air)
- [ ] B-roll layered over voiceover where appropriate
- [ ] Transitions smooth and intentional (not overdone)
- [ ] Color grading applied for consistent look
- [ ] Audio mixed and balanced (VO clear, music at appropriate level)

**Graphics & Text**:
- [ ] On-screen text overlays added (name/titles, stats, CTAs)
- [ ] Lower-thirds for speakers (if applicable)
- [ ] CTA button/end card added
- [ ] Logo animation (opening and closing)

**Captions**:
- [ ] Subtitles/captions added (SRT file or burned-in)
- [ ] Captions reviewed for accuracy (no auto-caption errors)

**Sound Design**:
- [ ] Background music added and mixed (not overpowering VO)
- [ ] Sound effects added where appropriate
- [ ] Audio levels normalized (no clipping or distortion)

**QA (Quality Assurance)**:
- [ ] Full video reviewed for errors (typos, visual glitches, audio issues)
- [ ] Timing validated (matches script and target length)
- [ ] Brand compliance check (logo, colors, messaging align with guidelines)
- [ ] Accessibility check (captions, color contrast)

## Review Checklist

### Content Quality

- [ ] **Message Clarity**: Video clearly communicates key message in target timeframe
- [ ] **Accuracy**: All product claims, stats, and information verified
- [ ] **Voice & Tone**: Aligns with BRD-002 (Voice & Tone Guide)
- [ ] **Engagement**: Opening hook grabs attention within 3 seconds
- [ ] **Pacing**: No dead air, smooth flow, appropriate speed
- [ ] **CTA**: Clear, specific, and easy to act on

### Visual Quality

- [ ] **Resolution**: HD or better (1080p minimum, 4K if budget allows)
- [ ] **Lighting**: Professional, consistent, no harsh shadows
- [ ] **Framing**: Subjects properly framed, rule of thirds applied
- [ ] **Color**: Color grading consistent, brand colors used appropriately
- [ ] **Graphics**: On-brand, high-quality, not pixelated or low-res
- [ ] **Text Readability**: On-screen text large enough, high contrast, visible on mobile

### Audio Quality

- [ ] **VO Clarity**: Voiceover clear, no background noise or echo
- [ ] **Music**: Background music enhances (not distracts), properly licensed
- [ ] **Sound Effects**: SFX add value (not overdone or cheesy)
- [ ] **Audio Mix**: VO louder than music, no clipping or distortion
- [ ] **Captions**: Accurate, synchronized, properly formatted

### Technical Specs

- [ ] **Aspect Ratio**: Correct for platform (16:9, 9:16, 1:1, etc.)
- [ ] **File Format**: MP4 (H.264 codec recommended for web)
- [ ] **File Size**: Optimized for platform (<500MB for most platforms)
- [ ] **Thumbnail**: Custom thumbnail created (high-quality, branded, compelling)

### SEO & Distribution

- [ ] **Title**: SEO-optimized, includes keyword, <60 characters
- [ ] **Description**: Detailed, includes keywords, links, timestamps
- [ ] **Tags**: Relevant keywords added (YouTube, if applicable)
- [ ] **Captions**: SRT file uploaded or captions enabled
- [ ] **UTM Parameters**: CTA links tagged for tracking
- [ ] **End Screen**: YouTube end screen configured (CTA, related videos, subscribe)

### Legal & Compliance

- [ ] **Music Licensed**: Background music licensed (not copyrighted)
- [ ] **Stock Footage**: Stock footage licensed or rights secured
- [ ] **Talent Release**: On-camera talent signed release form
- [ ] **Disclaimers**: Required disclaimers included (if making claims)
- [ ] **Accessibility**: Captions provided, WCAG compliant

## Agent Notes

### For Content Strategist
- Align video topic with broader content strategy and campaign calendar
- Ensure video supports buyer journey stage (awareness, consideration, decision)
- Coordinate distribution across channels for maximum reach
- Track performance and identify repurposing opportunities

### For Video Producer
- Prioritize clear messaging and pacing over production complexity
- Test script timing by reading aloud (adjust if too fast/slow)
- Ensure audio quality is excellent (bad audio kills good video)
- Design for sound-off viewing (captions, on-screen text)
- Optimize for mobile (large text, simple visuals)

### For Copywriter
- Write for the ear, not the eye (conversational, short sentences)
- Front-load key message (grab attention in first 3-5 seconds)
- Use active voice and direct language ("You can" not "Users are able to")
- Include specific, benefit-focused CTA
- Test script at 150 words/minute pace (2.5 words/second)
