## Purpose

This template provides a structured brief for creating infographics that transform complex data or processes into visually compelling, shareable content. Infographics drive engagement, social shares, and backlinks when designed with clear storytelling and data visualization principles.

## Review Checklist

### Content Accuracy

- [ ] **Data Verified**: All statistics double-checked against original sources
- [ ] **Sources Cited**: All data sources listed at bottom of infographic
- [ ] **Dates Confirmed**: Data recency noted (e.g., "2024 data")
- [ ] **Claims Substantiated**: No unsupported claims or misleading stats
- [ ] **Fact-Check**: SME or data analyst reviewed for accuracy

### Design Quality

- [ ] **Brand Compliance**: Follows brand guidelines (colors, fonts, logo usage)
- [ ] **Visual Hierarchy**: Clear flow, important info stands out
- [ ] **Readability**: Text is legible at all sizes (mobile and desktop)
- [ ] **Color Contrast**: WCAG AA compliant (4.5:1 for body text, 3:1 for large text)
- [ ] **Consistency**: Fonts, colors, spacing consistent throughout
- [ ] **Balance**: Good use of white space, not cramped or cluttered
- [ ] **Chart Accuracy**: Charts accurately represent data (no misleading visuals)

### SEO & Accessibility

- [ ] **Alt Text**: Descriptive alt text written for image (125 chars max)
- [ ] **File Name**: Descriptive file name with keyword (e.g., "social-media-stats-2025-infographic.png")
- [ ] **Captions**: Blog post includes text summary of infographic for SEO
- [ ] **Structured Data**: Blog post includes schema markup for infographic (JSON-LD)

### Legal & Compliance

- [ ] **Copyright**: Copyright notice included (© 2025 Company Name)
- [ ] **Permissions**: Rights secured for third-party data, images, icons
- [ ] **Attribution**: Sources properly attributed (no plagiarism)
- [ ] **Disclaimers**: Required disclaimers included if making claims
- [ ] **Trademark**: No unauthorized use of third-party trademarks

## Agent Notes

### For Content Strategist
- Source compelling data from whitepapers, blog posts, or original research
- Ensure infographic aligns with broader content strategy and SEO goals
- Coordinate launch with campaign calendar for maximum impact
- Track performance and identify repurposing opportunities

### For Graphic Designer
- Prioritize clarity over complexity (infographics should simplify, not overwhelm)
- Use visual hierarchy to guide reader through story (top to bottom)
- Ensure data visualizations are accurate (no misleading charts)
- Design for shareability (eye-catching, on-brand, mobile-friendly)
- Provide multiple formats for different distribution channels

### For Data Analyst
- Verify all statistics with original sources (no outdated or misattributed data)
- Ensure sample sizes and methodologies are sound
- Recommend appropriate chart types for each data point
- Flag any potentially misleading visualizations (e.g., truncated axes)
