## Purpose

This template provides a structured approach to creating SEO-optimized blog posts that align with brand voice, support content strategy goals, and drive measurable engagement. Suitable for thought leadership, educational content, product updates, and industry insights.

## Review Checklist

### Content Quality

- [ ] **Accuracy**: All facts, stats, and claims verified with citations
- [ ] **Originality**: Content passes plagiarism check (>95% unique)
- [ ] **Depth**: Topic covered comprehensively (vs. competitor content)
- [ ] **Clarity**: Written at appropriate reading level for audience
- [ ] **Voice & Tone**: Aligns with BRD-002 (Voice & Tone Guide)
- [ ] **Messaging**: Reinforces key messages from STR-002 (Messaging Matrix)
- [ ] **Grammar**: Passes grammar/spelling check (Grammarly, Hemingway, etc.)
- [ ] **Readability**: Flesch Reading Ease score 60+ (or appropriate for audience)

### SEO Optimization

- [ ] **Title Tag**: 55-60 characters, includes primary keyword
- [ ] **Meta Description**: 150-160 characters, includes keyword + CTA
- [ ] **URL Slug**: Short, descriptive, includes primary keyword
- [ ] **H1 Tag**: One H1 only, matches title or close variation
- [ ] **Header Hierarchy**: Logical H2/H3/H4 structure (no skipped levels)
- [ ] **Keyword Density**: Primary keyword 1-2%, natural placement
- [ ] **Keyword Variations**: Secondary and long-tail keywords included
- [ ] **Internal Links**: 3-5 relevant internal links with descriptive anchors
- [ ] **External Links**: 2-3 authoritative sources (open in new tab)
- [ ] **Image Alt Text**: All images have descriptive alt text with keywords
- [ ] **Image Optimization**: Images compressed (<200KB each)
- [ ] **Mobile-Friendly**: Formatted correctly for mobile devices
- [ ] **Page Speed**: Load time <3 seconds (test with PageSpeed Insights)

### Engagement Elements

- [ ] **Compelling Hook**: First 2-3 sentences grab attention
- [ ] **Scannable Format**: Short paragraphs (3-4 sentences), bullet points, subheads
- [ ] **Visual Variety**: 3-5 images, charts, or graphics
- [ ] **Data Visualization**: Stats presented visually where appropriate
- [ ] **Examples**: Real-world examples or case studies included
- [ ] **Quotes**: Expert or customer quotes add credibility
- [ ] **CTA Placement**: Primary CTA in mid-post and conclusion
- [ ] **CTA Clarity**: CTA is specific, actionable, and benefits-oriented
- [ ] **Social Sharing**: Social share buttons enabled
- [ ] **Related Content**: "Read Next" or related posts suggested

### Legal & Compliance

- [ ] **Attribution**: All sources, images, quotes properly attributed
- [ ] **Permissions**: Rights secured for all third-party content
- [ ] **Disclaimers**: Required disclaimers included (medical, financial, legal, etc.)
- [ ] **Privacy**: No PII exposed without consent
- [ ] **Accessibility**: WCAG 2.1 AA compliant (alt text, color contrast, etc.)
- [ ] **Copyright**: No copyrighted material used without permission
- [ ] **Claims Substantiation**: Product/service claims supported by evidence

### Technical Setup

- [ ] **UTM Parameters**: CTA links tagged for tracking
- [ ] **Schema Markup**: Article schema implemented (JSON-LD)
- [ ] **Open Graph Tags**: OG tags for social sharing (title, description, image)
- [ ] **Twitter Card**: Twitter card tags configured
- [ ] **Canonical Tag**: Canonical URL set to prevent duplicate content
- [ ] **Sitemap**: Added to XML sitemap
- [ ] **Indexing**: Set to "index, follow" (unless intentionally noindex)

## Agent Notes

### For Content Strategist
- Align post topic with content calendar (STR-007) and campaign strategy
- Ensure post supports broader content pillar and topic cluster strategy
- Coordinate with other content formats (e.g., repurpose into video, infographic)
- Track content performance against strategic goals

### For SEO Specialist
- Conduct thorough keyword research before drafting outline
- Analyze top-ranking content to identify gaps and differentiation opportunities
- Ensure technical SEO elements (schema, canonical, internal linking) are correct
- Monitor ranking progress and recommend optimizations

### For Copywriter
- Follow voice & tone guidelines from BRD-002
- Write compelling, scannable content (short paragraphs, subheads, bullets)
- Use storytelling techniques (anecdotes, examples, scenarios)
- Include data and expert quotes for credibility
- Write benefit-focused CTAs, not feature-focused
